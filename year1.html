2D Platformer

For My First Project I had to create a 2d platformer with the basic mechanics of walk, run, jump with collectable objects and jump pads I used premade art as Im not the greatest at 2D art so I mainly focused on the coding side of the project. I created a narrative system and custom boss fight with multiple attacks for this project. I enjoyed creating this game as I was able to expirement with different type of movement mechanics when chained together.
